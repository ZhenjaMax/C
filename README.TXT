1. Соберите программу. Используйте: g++ main.cpp

2. Ограничения программы.
Программа работает с BMP-изображениями, удовлетворяющие следующим условиям:
- формат изображения: BMP;
- сигнатура изображения: 0x4d32, 0x4349 или 0x5450;
- количество битов на точку изображения: 24;
- тип сжатия изображения: 0;
- размер полотна: не должен превышать 8192 по ширине или высоте.

3. Возможности программы.
1) Рисование окружности.
Пример использования: ./a.out -l white.bmp --circle 40,40,120,120 -t 30 -c orange -f black -s out1.bmp
2) Отражение указанной области изображения.
Пример использования: ./a.out --load marbles.bmp -s out2.bmp --mirror 50,50,300,400 --axis v
3) Копирование указанной области изображения.
Пример использования: ./a.out --save out3.bmp -l night.bmp --copy 200,250,300,400,400,400

Для вызова подробной справочной информации, используйте: ./a.out -h
